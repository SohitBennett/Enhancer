# Favicon Files

## Files Created:
- **favicon.svg** - Primary favicon (vector, scales perfectly)
- Used in index.html as the main favicon

## Design:
The logo features:
- Vibrant gradient background (blue → purple → pink) matching the app's color scheme
- A stylized photo/image icon with yellow-orange gradient
- Enhancement sparkles and arrow indicating AI enhancement
- Bold black borders matching the retro Windows 95 aesthetic
- Clean, geometric design that's recognizable even at small sizes

## Colors Used:
- Background: Blue (#2563eb) → Purple (#9333ea) → Pink (#ec4899)
- Image frame: Yellow (#fbbf24) → Orange (#f59e0b)
- Sparkles/Effects: Light yellow (#fef08a)
- Borders: Black (#000)
